# Cesium_Vue3_Ts 初始化项目模板框架

基于 vite4.0
使用 cesium、echarts、pinia、sass、element-plus 等相关技术栈，element 已实现自动化按需导入配置。
使用 scss 样式处理器进行样式编写，使用 rem 进行响应式配置。
实现 Cesium 适配 ts、屏幕自适应！
实现Gzip代码压缩、路由懒加载、Cesium外部化引入，提高项目加载速度，打包体积！

### 安装依赖

```sh
pnpm install
```

### 运行项目

```sh
pnpm dev
```

### 打包项目

```sh
pnpm build
```

### 预览 打包后代码

```sh
pnpm preview
```
